Le projet est découpé en 2 parties: 
    
    1. La simulation de la grotte dans une console
        Pour lancer cette simulation, rendez-vous le fichier src/main/java/com/projet/grotte/algorithm/CaveSimulation.java et eécutez la méthode main()

    2. La simulation de la grotte dans une interface graphique
        Pour lancer cette simulation, rendez-vous le fichier src/main/java/com/projet/grotte/design/CaveSimulation.java et eécutez la méthode main()

            - les gouttes sont de couleur bleu
            - les stalactites en couleur "brique"
            - les stalagmites en couleur "brique"
            - les colonnes sont rouge
            - les draperies sont rouges foncées

Il y a également des tests.
